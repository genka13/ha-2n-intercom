# 2N Intercom (Home Assistant Custom Integration)

Custom integration for 2N intercom devices (e.g., 2N IP Style / 2N IP Verso 2 / 2N IP One) using the 2N HTTP API and RTSP.

This repository is prepared for installation via **HACS** (custom repository).

## Installation (HACS)

1. In Home Assistant open **HACS → Integrations → ⋮ → Custom repositories**
2. Add your GitHub repository URL and select category **Integration**
3. Install **2N Intercom**
4. Restart Home Assistant
5. Add the integration: **Settings → Devices & services → Add integration → 2N Intercom**
